# Smart Soil Moisture Monitor

## Project Overview
This project monitors soil moisture using Arduino Uno and a soil moisture sensor.

## Components Used
- Arduino Uno
- Soil Moisture Sensor
- LEDs
- Breadboard

## Features
- Detects dry and wet soil
- Red LED for dry soil
- Green LED for wet soil
- Data visualization using ThingSpeak

## Software Used
- Tinkercad
- Arduino IDE
- ThingSpeak

## Applications
- Smart irrigation
- Agriculture monitoring
- Garden automation
